## Description

**e-wallet(이하, 간편결제)** 서비스에서 제공하는 멤버쉽 서비스로서,<br/>
간편 결제 서비스에 가입하는 개인/법인 고객의 정보를 **소유**하고 **관련 정보의 변경에 대한 의무**를 가진 서비스

## MVP Version

- 개인/법인 고객의 정보 소유
- 멤버쉽(개인/법인) 신규 가입
- 개인/법인 고객의 정보 조회
