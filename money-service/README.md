## Description

**e-wallet(이하, 간편결제)** 서비스에서 제공하는 머니 서비스로서,<br/>
고객의 선불충전금액의 **증액/감액**을 담당하고 이를 **기록**하는 의무를 가진 서비스

## MVP Version

- [membership-service](../membership-service) 및 [banking-service](../banking-service)를 이용하여 금액을 충전하는 기능
- 충전 내역 조회 기능
