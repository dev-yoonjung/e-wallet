## Description

**e-wallet(이하, 간편결제)** 서비스에서 제공하는 뱅킹 서비스로서,<br/>
외부 은행과의 직접적인 통신을 담당하고, 펌뱅킹 계약이나 수수료 관리 등 **외부 은행 사용과 관련된 모든 기능**을 제공하는 서비스

## MVP Version

- 외부 은행과 입출금 요청 펌뱅킹 통신 담당
- 내부 고객의 계좌 정보 등록
